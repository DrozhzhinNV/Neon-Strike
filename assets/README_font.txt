Поместите сюда файл font.ttf — любой свободный шрифт.

Варианты:
  - Скачайте DejaVuSans.ttf с https://dejavu-fonts.github.io/
  - Скопируйте C:\Windows\Fonts\arial.ttf в эту папку и переименуйте в font.ttf

Если файл отсутствует, игра будет использовать Arial автоматически,
но только если он установлен в системе. Шрифт нужен только для текста HUD.
